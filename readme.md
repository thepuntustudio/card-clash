# ⚔️ Card Clash

A turn-based card battler where you Attack, Block, or Heal to defeat increasingly powerful enemies.

## 🎮 Gameplay

Choose your action each turn:
- **⚔️ Attack** – Deal 10 damage to the enemy
- **🛡️ Block** – Reduce incoming damage by 8
- **❤️ Heal** – Restore 15 HP

After your turn, the enemy strikes back! Each victory makes the next enemy stronger.

## 🚀 Getting Started

### Prerequisites
- Unity 2022.3 or newer

### Clone & Open

```bash
git clone https://github.com/thepuntustudio/card-clash.git
```

1. Open Unity Hub
2. Click Add → select the project folder
3. Open the project
4. Press Play in the Unity Editor

## 🏗️ Building
1. Go to File > Build Settings
2. Select your target platform
3. Click Switch Platform (if needed)
4. Click Build
